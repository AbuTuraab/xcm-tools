## Usage

Install package:

```sh
# npm
npm install @paraspell/sdk

# yarn
yarn install @paraspell/sdk

# pnpm
pnpm install @paraspell/sdk
```

Import:

```js
// ESM
import { } from '@paraspell/sdk'

// CommonJS
const { } = require('@paraspell/sdk')
```
